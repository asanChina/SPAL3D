Detailed Description

ArcInfoASCIIGenerator is inherited from DataGenerator, responsible for generating ARC/INFO ASCII data file.

This class implements "DataGenerator::generate(...)" virtual function, also including some self-declared member functions for setting/getting format specific infomation.
